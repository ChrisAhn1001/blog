```bash
git init: **새로운 Git 저장소(repository)를 생성할 때 사용하는 Git 명령어**

git status: 전체적인 상황

git commit -m :메세지 설정

git push origin master : **origin이라는 remote로 현재 로컬 브랜치중 my-branch라는 것을 push하겠다는 의미** 이다.```
